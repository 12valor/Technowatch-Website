# TechnoWatch Design System

## Design Philosophy

**Visual Language**: Modern, professional, and technologically sophisticated. The design embodies innovation through clean geometric forms, purposeful animations, and a balanced color palette that conveys trust and cutting-edge expertise.

**Color Palette**: 
- Royal Blue: #1e3a8e (Primary header color)
- Deep Blue: #1e40af (Secondary blue)
- Accent Blue: #3b82f6 (Interactive elements)
- Light Blue: #60a5fa (Hover states)
- Pure White: #ffffff (Background and text)
- Light Gray: #f8fafc (Section backgrounds)
- Medium Gray: #64748b (Secondary text)
- Dark Gray: #334155 (Primary text)
- Success Green: #10b981 (Success states)
- Warning Orange: #f59e0b (Warning states)

**Typography**:
- Display Font: "Inter" - Modern, clean sans-serif for headings
- Body Font: "Inter" - Consistent typography hierarchy
- Font Weights: 300 (light), 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
- Heading Sizes: 48px (h1), 36px (h2), 24px (h3), 20px (h4)
- Body Size: 16px (base), 14px (small)

## Visual Effects

**Used Libraries**:
- Anime.js: For smooth micro-animations and element transitions
- Splide.js: For image carousels and product galleries
- Custom CSS: For hover effects and micro-interactions

**Animation Principles**:
- Smooth fade-ins with 300ms duration
- Subtle hover effects with 200ms transitions
- Staggered reveals for card grids
- Color transitions for interactive elements
- Scale transforms for button interactions

**Header Effect**: 
Clean royal blue navigation bar with subtle shadow and backdrop blur effect for modern depth.

**Interactive Elements**:
- 3D tilt effects on hover for cards and images
- Color morphing for buttons and links
- Smooth scale transforms (1.05x) on hover
- Subtle glow effects for featured elements

## Component Styling

**Navigation Bar**:
- Royal blue background (#1e3a8e)
- White text with hover transitions
- Fixed position with backdrop blur
- Mobile-responsive hamburger menu

**Buttons**:
- Primary: Royal blue background with white text
- Secondary: White background with royal blue border
- Hover: Subtle scale and shadow effects
- Border radius: 8px for modern appearance

**Cards**:
- White background with subtle shadows
- Hover: Lift effect with enhanced shadow
- Border radius: 12px for premium feel
- Smooth transitions for all interactions

**Forms**:
- Clean white inputs with blue focus states
- Floating labels with smooth transitions
- Validation states using color coding
- Modern input styling with subtle shadows

## Responsive Design

**Breakpoints**:
- Mobile: 320px - 768px
- Tablet: 768px - 1024px  
- Desktop: 1024px+

**Mobile Adaptations**:
- Touch-friendly 44px minimum tap targets
- Simplified animations for performance
- Stacked layouts for complex components
- Optimized typography scaling

## Accessibility

**Color Contrast**: All text maintains 4.5:1 contrast ratio minimum
**Focus States**: Clear blue outline indicators
**Motion**: Respects prefers-reduced-motion settings
**Screen Readers**: Proper ARIA labels and semantic HTML
